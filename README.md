# digital_clock_python_using_Tkinter
Overview:
This project is a Digital Clock Application developed using Python and the Tkinter GUI framework. It displays real-time system time and date, updating automatically every second. The application demonstrates concepts of GUI development, event-driven programming, and real-time data handling.

* Key Features
Real-time Digital Clock Display
Automatic update every second using event loop
Displays Hours, Minutes, Seconds, AM/PM, and Date
Lightweight and user-friendly interface
Built using standard Python libraries
🛠️ Tech Stack
Programming Language: Python
GUI Framework: Tkinter
Library Used: Time (strftime function)
📂 Project Structure
digital-clock/
│── clock.py
│── README.md

▶️ Installation and Execution
Install Python (3.x recommended)
Clone the repository:
git clone https://github.com/your-username/digital-clock.git
Navigate to project directory:
cd digital-clock
Run the application:
python clock.py

⚙️ Working Principle
The application uses:

Tkinter Label Widget to display time
strftime() function to fetch formatted system time
after() method to update the clock every 1000 milliseconds

This ensures continuous real-time updates without freezing the GUI.
